# RCA Security Group Website

A stunning, animated cybersecurity consulting website featuring an epic 30-second demo animation that showcases real cyber threats and RCA Security Group's response capabilities.

## 🚀 Features

- **Epic Demo Animation** - 30-second cinematic experience showing a real cyber attack and RCA's response
- **Realistic Cybersecurity Elements** - Uses actual tools, CVEs, and terminology
- **Professional Design** - Modern, responsive design with advanced animations
- **Smart Demo System** - New users see the demo, returning users skip directly to content
- **Victory Page** - Dedicated page showcasing RCA's value propositions
- **Complete Service Portfolio** - Detailed cybersecurity services and pricing

## 🎬 Demo Animation

The website features a dramatic 30-second animation that:
1. Shows a normal corporate website
2. Demonstrates a realistic cyber attack using actual tools (Hydra, CVE-2021-44228)
3. Shows data breach in progress with realistic numbers
4. Demonstrates RCA Security Group's professional response
5. Transitions to a victory page highlighting RCA's capabilities

## 🛠️ Technology Stack

- **HTML5** - Semantic structure and modern markup
- **CSS3** - Advanced animations, gradients, and responsive design
- **JavaScript** - Interactive elements and animation control
- **GitHub Pages** - Hosting and automatic deployment
